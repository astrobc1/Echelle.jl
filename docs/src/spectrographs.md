Spectrographs
=============

The following spectrogaphs are currently supported with different levels of implementation.

# iSHELL

Support status:
- Reduction of data in KGAS mode with a 0.375'' slit (tested).
- Generation of RVs in KGAS mode using the 13CH4 gas cell as a wavelength reference and LSF constraint (tested).

# PARVI

Support status:
- Reduction of data in the static mode.
- Wavelength calibration using the laser frequency comb.
- Generation of RVs.

# CHIRON

Support status:
- Generation of RVs in slicer mode and high-resolution modes using the iodine gas cell (tested) as a wavelenght reference and LSF constraint (tested).

# MINERVA

Support status:
- Generation of RVs using the provided ThAr wavelength solutions.

# HARPS-S

Support status:
- Generation of RVs in assuming they are formatted in the same way as provided on the archive.

# ESPRESSO

Support status:
- Generation of RVs in assuming they are formatted in the same way as provided on the archive.

# NIRSPEC

Support status:
- Reduction of data in K-band mode (tested, still preliminary).
- Generation of RVs in assuming they are formatted in the same way as provided on the archive.
